# Bigfish 组件开发 TypeScript 脚手架

## 准备工作

- 安装 [nodejs](https://nodejs.org/en/)
- 使用最新版本的 [tnpm](http://web.npm.alibaba-inc.com/)

## 开发

```bash
# 安装依赖
$ tnpm install

# 开发 library
$ tnpm run dev

# 打包 library
$ tnpm run build
```
